# Backend | FastAPI

## Overview

The backend is a FastAPI service that exposes the project API, database-backed
simulation endpoints, and model inspection routes.

Base URL:

```text
http://localhost:8000
```

Swagger:

```text
http://localhost:8000/docs
```
