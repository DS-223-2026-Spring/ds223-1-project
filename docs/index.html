Campaign Optimization Engine

An AI-driven system that uses contextual bandits to select the optimal marketing action for each customer, learning from outcomes over time.

DS 223 — Marketing Analytics — Spring 2026

Overview

Traditional marketing relies on static segmentation: group customers by RFM scores, assign a blanket promotion, and hope for the best. This system replaces that approach with a contextual bandit (LinUCB) that learns which promotional action works best for each customer profile — balancing exploration of new strategies with exploitation of proven ones.

Architecture
